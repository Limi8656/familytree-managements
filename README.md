You can use it to:

- Add family members

- Track life status
- Manage more than one family branch

## 🗂️ How to use it

When the app opens, start with one family member and build from there.

### Add a person

- Enter the person’s name
- Add birth details if you know them
- Set their life status
- Add a short note if needed

### Link family members

- Connect parents and children
- Mark partners or spouses
- Keep branches grouped by family line
- Review the tree to check each link

### Track life status

Use the life status field to record common states such as:

- Living
- Deceased
- Unknown
- Separated
- Other family notes

### Manage more than one family group

If you work with several family lines, you can keep them separate and easy to find. This helps when you are building trees for large families or collecting records for different sides of the family.
